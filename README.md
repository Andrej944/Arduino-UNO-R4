## 🌐 About Arduino UNO R4

The **Arduino UNO R4** is a modern upgrade of the classic UNO board, featuring:

- 32-bit ARM Cortex-M4 microcontroller
- Increased memory and processing power
- Improved peripheral support (DAC, CAN, OPAMP, etc.)
- USB-C connectivity
- Optional WiFi + Bluetooth (UNO R4 WiFi version)

It enables more advanced embedded and IoT-style projects while keeping the familiar UNO form factor.
